
Keyboard Shortcuts

Enter 		open selected item  
Backspace 	go to previous directory  
Delete 		delete selected items  
Esc 		de-select items or return to homescreen  
F2   		rename  
Ctrl-1 		Icon view  
Ctrl-2 		list view  
Ctrl-3 		toggle preview pane  
Ctrl-4 		sort by name  
Ctrl-5 		sort by size (including directory sizes)  
Ctrl-6 		sort by date  
Ctrl-C 		Copy  
Ctrl-V 		Paste  
Ctrl-N 		new window
